# ⚡ Sửa lỗi nhanh - Unknown at rule @tailwind

## 🎯 Lỗi: "Unknown at rule @tailwind" trong file CSS

### ✅ CÁCH SỬA - Chọn 1 trong 2:

---

### **Cách 1: Cài Extension (Khuyến nghị)** ⭐⭐⭐

1. **Ctrl + Shift + X** (mở Extensions)
2. Tìm: **"Tailwind CSS IntelliSense"**
3. Click **Install**
4. **Ctrl + Shift + P** → gõ **"Reload Window"**

✅ Xong! Warning sẽ mất và có thêm autocomplete cho Tailwind classes!

---

### **Cách 2: Tắt CSS Validation (Nhanh)** ⚡

1. **Ctrl + Shift + P** (Command Palette)
2. Gõ: **"Developer: Reload Window"**
3. Enter

✅ File `.vscode/settings.json` đã được tạo sẵn và sẽ tự động tắt warning!

---

## 💡 Giải thích:

- `@tailwind` là directive của Tailwind CSS
- VS Code mặc định không hiểu Tailwind directives
- Đây chỉ là **WARNING**, không phải lỗi
- Code vẫn chạy bình thường!
